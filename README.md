# Gravity Flip Runner

Endless runner where you flip gravity to dodge obstacles. Simple but addictive.

## Play Now

[**Play in Browser**](https://gravity-flip-runner.pages.dev)

## Running Locally

Just open `index.html` in your browser. No build step, no dependencies, no nonsense.

If you want to use a local server:

```bash
python -m http.server 8000
```

Then hit `http://localhost:8000`

## Controls

- **Space** or **Tap** - Flip gravity
- That's the only control

## Tips

- Getting close to obstacles without hitting them gives you bonus points
- Orb combos are worth way more than individual collects
- Game speeds up over time. A lot.

## Built With

- Vanilla JS
- HTML5 Canvas
- CSS

No frameworks. No bundlers. Just three files.
